umbral playground

The Vite-built React frontend isn't in dist/ yet. You're seeing this placeholder because build.rs couldn't produce the bundle.

To enable the full UI

  1. Install Node.js 20+ (nodejs.org) so npm is on your PATH.
  2. Install the frontend's dev dependencies (one-time setup): 
    cd plugins/umbral-playground/frontend
npm install
  3. Rebuild the crate. build.rs will run npm run build automatically and copy the Vite output into plugins/umbral-playground/dist/: 
    cargo build -p umbral-playground

Common failure modes

  1. npm not in PATH — install Node.js.
  2. frontend/node_modules missing — run npm install once.
  3. Vite build itself errored — scroll up in the cargo output for the actual error.

The crate compiles + serves this placeholder either way — only the React UI is gated.
