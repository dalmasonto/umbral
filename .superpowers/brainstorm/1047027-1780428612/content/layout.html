Playground main layout

Pick the shape that feels right for an API playground. All three are usable; the question is which one you'd want to use daily.

A

Three-pane (Postman / Insomnia style)

Left: endpoint tree. Center: request builder. Right: response viewer. Familiar, dense, no scrolling to find anything. Best for power users.

POST /api/article/
GET /api/article/{id}/
PUT /api/article/{id}/
...

B

Swagger-style stacked

Endpoint list on top, vertically-stacked request/response sections per endpoint. Easier first impression, more scrolling, easier to print/share a single endpoint.

C

Two-pane split (minimal)

Left: collapsible endpoint list. Right: full-height builder + response stacked. Calmest UI, easiest to focus on one request at a time.
