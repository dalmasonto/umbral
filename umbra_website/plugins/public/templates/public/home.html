framework dossier

Umbra is a Rust framework for building whole web applications.

It gives Rust projects an app shape: models, migrations, templates, forms, admin, REST, OpenAPI, security defaults, and project-owned plugins. The important part is that the batteries are still plugins, not hidden framework magic.
