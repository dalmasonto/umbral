A complete app framework

Not just HTTP routing. Models, migrations, an admin, forms, REST and background work — the productive shape you already know.

Batteries are plugins

Official capabilities ship as prebuilt plugins. Enable them, replace them with a community plugin, or extend them — same boundary either way.

Proven by this site

Umbra.rs is itself an Umbra project — the directory, reviews, discussions and admin you see are the same systems it documents.
