Changelog

What's shipped, what's next

Umbra is greenfield and moving fast. Here's where it stands today and the milestones on the way to v0.1.

Current
v0.0.1

The core loop, end to end

  • ORM with managed migrations — declare → migrate → change → migrate.
  • Auto admin: CRUD, search, multi-filters, FK/M2M/O2O pickers, permissions, dashboard widgets.
  • REST + OpenAPI + an interactive playground; auth-gated resources.
  • Auth, sessions, OAuth (Google/GitHub) + account connection.
  • Secure-by-default: CSRF, HSTS, clickjacking headers, auto-escaping.
  • File/image fields with pluggable storage; soft deletes; signals; fixtures.
Roadmap
toward v0.1
  • Email sending (SMTP / API backends) and a hardened background task queue.
  • WebSockets / SSE for realtime push — user- and room-targeted.
  • REST nested writable serializers; CSV / Excel import-export.
  • A testing & factory library for models and pages.
  • Caching, rate limiting, structured logging, and metrics.

Track the work on GitHub, or follow releases via the newsletter.
