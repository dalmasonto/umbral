What is already wired

ORM & models

Project, Task and Comment live in plugins/projects/. Change one, then run makemigrations.

Migrations

Autodetected from your models. The declare → migrate → change → migrate loop is the whole product.

Admin

CRUD for every registered model at /admin/. Run createsuperuser to get in.

REST API

Auto-exposed JSON at /api/task, plus a console at /api/playground.

Auth & sessions

Users, password hashing, sessions, CSRF. The endpoints are ours; the pages are yours.

Plugins

Everything above is a plugin. So is the app you are about to write — umbral startapp blog.
