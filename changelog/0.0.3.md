# 0.0.3 — 2026-06-29

The **auth release**. A complete authentication surface lands: email verification, password forgot/reset, form-action routes (form in → 303 redirect out) alongside the JSON API, and a pluggable mailer seam. REST gains read-only view scoping, sessions gain log-out-everywhere, and the playground persists your work.

- **auth** — form-action auth endpoints (form in, 303 redirect out) via with_form_routes
