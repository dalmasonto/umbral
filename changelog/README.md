# Changelog

Release notes for **umbral**, one file per version. umbral publishes all crates together in versioned rounds; each file below covers a whole round (not a single crate). For the exact per-crate bullet history that these are distilled from, see each crate's own `CHANGELOG.md` (generated by release-plz from conventional commits).

Versions follow [Semantic Versioning](https://semver.org/); the format is inspired by [Keep a Changelog](https://keepachangelog.com/).
