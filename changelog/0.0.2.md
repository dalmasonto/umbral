# 0.0.2 — 2026-06-26

A **packaging & polish** release. The framework becomes properly consumable from crates.io — scaffolded projects depend on published versions, every crate carries a README, and the documentation site goes live. The last of the "prior-art framing" is removed from the codebase so umbral stands on its own.

## Added

- **cli** — scaffold projects with crates.io version deps, not git paths

## Changed

- **docs** — deploy the Specra documentation site to GitHub Pages
- Add per-crate READMEs for crates.io, and link them to the documentation site
- Add health + oauth plugin doc pages, and repoint their READMEs
- Remove the prior-art (Django) framing across the codebase, crate metadata, and code
